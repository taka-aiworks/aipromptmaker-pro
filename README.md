# AI Prompt Maker

Stable Diffusion用の高機能プロンプト生成ツール。LoRA学習データセット作成から大量生成、漫画風表現まで対応。

## 🚀 主な機能

- **📚 学習モード**: LoRA学習用データセット生成（配分ルール最適化）
- **🏭 量産モード**: 大量バリエーション生成（プリセット対応）
- **🎭 漫画モード**: 漫画・アニメ表現特化（商用LoRA対応）
- **📷 撮影モード**: 写真撮影風設定
- **🔍 単語モード**: タグ検索・選択機能
- **💾 プリセット**: 設定保存・読込・管理

## 📋 対応フォーマット

- AUTOMATIC1111 (Web UI)
- ComfyUI
- InvokeAI
- SD.Next
- NovelAI

### 1. 基本情報設定
```
キャラクター名: your_character

年齢・性別・髪型・服装・色を設定
```

### 2. モード選択

#### 📚 学習モード（LoRA訓練用）
- 24枚推奨
- 自動配分ルール適用
- キャプション同時生成

#### 🏭 量産モード（大量生成）
- プリセット選択（服装/表情/ミックス）
- 50枚推奨
- バリエーション設定

#### 🎭 漫画モード（表現豊富）
- 感情・表情プリセット
- 商用LoRA統合
- 2人キャラ対応

### 3. 生成・出力
- プロンプト確認
- CSV/テキスト出力
- 各種フォーマット対応

## ⚙️ 主要設定

### 基本情報
- **キャラ**: 名前、LoRAタグ、年齢、性別
- **外見**: 髪型、目、肌色（カラーホイール）
- **服装**: ワンピース/セパレート、色指定
- **保存**: JSONエクスポート/インポート

### 学習モード設定
```
📊 配分ルール（最適化済み）:
- three-quarter view: 45-50%
- bust/waist up: 54-66%
- smiling: 25-30%
- neutral: 50-58%
```

### 量産モード設定
```
👕 服装バリエーション: 服装変更、表情固定
🎭 表情バリエーション: 表情変更、服装固定
🎨 ミックス: 両方変更
⚙️ カスタム: 手動設定
```

### 漫画モード設定
```
😊 SFW感情プリセット:
喜び・悲しみ・怒り・恥ずかしがり・驚き等

🔞 NSFWシチュプリセット:
ロマンス・お風呂・ナイト・コスプレ等
```

## 🎨 NSFW対応

### レベル設定
- **L1 (R-15)**: 軽微な表現
- **L2 (R-18)**: 成人向け表現  
- **L3 (R-18G)**: 過激な表現

### 要素選択
- 表情・露出・シチュエーション
- ポーズ・衣装・ライティング
- 身体特徴・アクセサリー
